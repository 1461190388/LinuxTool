定义函数
========
为了实现模块化编程，我们可以把m文件中的某些代码抽象成一个函数。编写自定义函数也是在单独的一个m文件中完成的。
--------
  与其他语言不同，MATLAB支持函数有多个返回值。

##一个返回值
```matlab
function c = add(a,b)
  c = a+b;
end 
```
>MATLAB中定义函数时无须使用return语句。
>>然而同其他语言一样，return同样可用来终止函数调用。

```matlab
function c = divide(a,b)
  if b == 0
      disp('error')
      return
  end
  c = a/b;
end
```
###多个返回值
```matlab
function [c,d] =  fun(a,b)
  c = a+b;
  d = a*b;
end
```
