##邻接表

###链式前向星
链式前向星是邻接表的静态实现。
```cpp
int head[n];   //head[i]=j,表示以i为起点的第一条边在Edges数组中的位置是j
struct EdgeNode
{
    int to;    //该边指向的终点
    int w;     //权重
    int next;  //同起点的下一条边
};
EdgeNode Edges[n];
```

>链式前向星并非专业术语。是网友的说法，但是在中国也比较流行。
